Дані часових рядів

Вступ

До цього моменту ми зосереджувалися на перехресних даних.

  • Вибірка з сукупності (наприклад, люди, округи, країни).
  • Вибірка в один момент часу (наприклад, 1 січня 2015 р.).
  • У нас було \(n\) індивідуумів, кожен з індексом \(i\) у \(\left\{1,\,\ldots,\, n \right\}\).

Сьогодні ми зосередимося на іншому типі даних: дані часових рядів.

  • Вибірка в межах однієї одиниці/особи (наприклад, Київ).
  • Декілька спостережень для тієї самої одиниці (наприклад, Київ: 1990–2020).
  • У нас є \(\color{#e64173}{T}\) часових періодів, кожен з яких індексований \(t\) у \(\left\{1,\,\ldots,\, T \right\}\).

Моделі часових рядів

Вступ

Наша модель зараз виглядає приблизно так

\[ \begin{align} \text{Births}_t = \beta_0 + \beta_1 \text{Income}_t + u_t \end{align} \]

або можливо так

\[ \begin{align} \text{Births}_t = \beta_0 + \beta_1 \text{Income}_t + \beta_2 \color{#e64173}{\text{Income}_{t-1}} + u_t \end{align} \]

або навіть так

\[ \begin{align} \text{Births}_t = \beta_0 + \beta_1 \text{Income}_t + \color{#e64173}{\beta_2 \text{Income}_{t-1}} + \beta_3 \color{#6A5ACD}{\text{Births}_{t-1}} + u_t \end{align} \]

де \(t-1\) позначає період часу до \(t\) (відставання (анг. lagged) доходу або народжуваності).

Припущення

  1. New: Слабко стійкі входи — по суті, \(x_{t+k}\) у періоді \(t+k\) слабо корелює з періодом \(x_t\) (коли \(k\) “велике”).

  2. \(y_t\) є лінійною функцією своїх параметрів і збурення.

  3. У наших даних немає ідеальної колінеарності.

  4. \(u_t\) має умовне середнє нульове (екзогенність), \(\mathop{\boldsymbol{E}}\left[ u_t \middle| X \right] = 0\).

  5. \(u_t\) є гомоскедастичним з нульовою кореляцією між \(u_t\) і \(u_s\), тобто, \(\mathop{\text{Var}} \left( u_t | X \right) = \mathop{\text{Var}} \left( u_t \right) = \sigma^2\) та \(\mathop{\text{Cor}} \left( u_t,\,u_s \middle| X \right) = 0\).

  6. Нормальність збурень, тобто, \(u_t\overset{\text{iid}}{\sim}\mathop{N}\left( 0,\,\sigma^2 \right)\).

Параметри моделі

Моделювання часових рядів зводиться до двох класів моделей.

  1. Статичні моделі: Не допускають персистентного ефекту.

  2. Динамічні моделі: Допускають персистентний ефект.

  • Моделі з пояснювальними змінними
  • Моделі авторегресії, розподіленої затримки (анг. Autoregressive, distributed-lag, ADL).

Параметри моделі

Варіант 1: Статичні моделі

Статичні моделі припускають, що результат залежить від лише поточного періоду.

\[ \begin{align} \text{Births}_{\color{#e64173}{t}} = \beta_0 + \beta_1 \text{Income}_{\color{#e64173}{t}} + u_{\color{#e64173}{t}} \end{align} \]

Тут ми повинні вірити, що дохід відразу впливає на народжуваність і не впливає на народжуваність у майбутньому.

Ми також повинні вірити, що народжуваність не залежить від попередніх показників народжуваності.

Може бути дуже обмеженим способом розгляду даних часових рядів.

Параметри моделі

Варіант 2: Динамічні моделі

Динамічні моделі дозволяють результату залежати від інших періодів.

Параметри моделі

Варіант 2a: Динамічні моделі з лаговими пояснювальними змінними

Ці моделі дозволяють результату залежати від пояснювальної змінної (змінних) в інші періоди.

\[ \begin{align} \text{Births}_{\color{#e64173}{t}} = &\beta_0 + \beta_1 \text{Income}_{\color{#e64173}{t}} + \beta_2 \text{Income}_{\color{#6A5ACD}{t-1}} + \\ &\beta_3 \text{Income}_{\color{#6A5ACD}{t-2}} + \beta_4 \text{Income}_{\color{#6A5ACD}{t-3}} + u_{\color{#e64173}{t}} \end{align} \]

Тут дохід негайно впливає на кількість народжень і впливає на майбутню кількість народжень.

Іншими словами: народжуваність сьогодні залежить від сьогоднішнього доходу та відставання доходу—наприклад, дохід минулого місяця, дохід минулого року, …

Оцініть загальні ефекти шляхом підсумовування коефіцієнтів лагів, наприклад, \(\beta_1 + \beta_2 + \beta_3 + \beta_4\).

Примітка: Ми все ще припускаємо, що поточна народжуваність не впливає на майбутню народжуваність

Параметри моделі

Варіант 2б: Динамічні моделі з авторегресією, розподіленої затримки (ADL)

Ці моделі дозволяють результату залежати від пояснювальної(их) змінної(й) та/або змінної результату в попередніх періодах.

\[ \begin{align} \text{Births}_{\color{#e64173}{t}} = \beta_0 + \beta_1 \text{Income}_{\color{#e64173}{t}} + \beta_2 \text{Income}_{\color{#6A5ACD}{t-1}} + \beta_3 \text{Births}_{\color{#6A5ACD}{t-1}} + u_{\color{#e64173}{t}} \end{align} \]

Тут поточний дохід впливає на поточну народжуваність та майбутню народжуаність.

Крім того, поточна народжуваність впливає на майбутню народжуваність — ми допускаємо затримки змінної результату.

Порядок лагів

Моделі ADL часто вказуються як \(\text{ADL}(\color{#FFA500}{p},\,\color{#e64173}{q})\), де

  • \(\color{#FFA500}{p}\) — це (максимальна) кількість лагів для змінна результату.

  • \(\color{#e64173}{q}\) — це (максимальна) кількість лагів для пояснювальні змінні.

Приклад: \(\text{ADL}(\color{#FFA500}{1},\,\color{#e64173}{0})\)

\[ \begin{aligned} \text{Births}_t = \beta_0 + \beta_1 \text{Income}_t + \beta_2 \text{Births}_{\color{#FFA500}{t-1}} + u_t \end{aligned} \]

Приклад: \(\text{ADL}(\color{#FFA500}{2},\,\color{#e64173}{2})\)

\[ \begin{aligned} \text{Births}_t = &\beta_0 + \beta_1 \text{Income}_t + \beta_2 \text{Income}_{\color{#e64173}{t-1}} + \beta_3 \text{Income}_{\color{#e64173}{t-2}} \\ & + \beta_4 \text{Births}_{\color{#FFA500}{t-1}} + \beta_5 \text{Births}_{\color{#FFA500}{t-2}} + u_t \end{aligned} \]

Складність

Через свої затримки моделі ADL фактично оцінюють навіть складніші зв’язки, ніж ви могли спочатку подумати

Розглянемо ADL(1, 0): \(\text{Births}_t = \beta_0 + \beta_1 \text{Income}_t + \beta_2 \text{Births}_{t-1} + u_t\)

Випишіть модель для періоду \(t-1\):

\[ \begin{align} \text{Births}_{t-1} = \beta_0 + \beta_1 \text{Income}_{t-1} + \beta_2 \text{Births}_{t-2} + u_{t-1} \end{align} \]

яку ми можемо замінити на \(\text{Births}_{t-1}\) у першому рівнянні, тобто,

\[ \begin{align} \text{Births}_t = &\beta_0 + \beta_1 \text{Income}_t + \\ &\beta_2 \underbrace{\left( \beta_0 + \beta_1 \text{Income}_{t-1} + \beta_2 \text{Births}_{t-2} + u_{t-1} \right)}_{\text{Births}_{t-1}} + u_t \end{align} \]

Складність

Продовження…

\[ \begin{align} \text{Births}_t = &\beta_0 + \beta_1 \text{Income}_t + \\ &\beta_2 \underbrace{\left( \beta_0 + \beta_1 \text{Income}_{t-1} + \beta_2 \text{Births}_{t-2} + u_{t-1} \right)}_{\text{Births}_{t-1}} + u_t \\ =& \beta_0 \left(1 + \beta_2 \right) + \beta_1 \text{Income}_t + \beta_1 \beta_2 \text{Income}_{t-1} + \\ &\beta_2^2 \text{Births}_{t-2} + u_{t} + \beta_2 u_{t-1} \end{align} \]

Тоді ми могли б підставити в рівняння \(\text{Births}_{t-2}\), \(\text{Births}_{t-3}\), …

Зрештою ми досягаємо

\[ \begin{align} \text{Births}_t = &\beta_0 \left( 1 + \beta_2 + \beta_2^2 + \beta_2^3 + \cdots \right) + \\ &\beta_1 \left( \text{Income}_t + \beta_2 \text{Income}_{t-1} + \beta_2^2 \text{Income}_{t-2} + \cdots \right) +\\ & u_t + \beta_2 u_{t-1} + \beta_2^2 u_{t-2} + \cdots \end{align} \]

Включаючи лише один лаг залежної змінної, як у ADL(1, 0), ми неявно включаємо багато лагів пояснювальних змінних і збурень

Модель часткового коригування

Бувають випадки, коли насправді хочеться моделювати бажане значення спостереження, а не її фактичне значення, але ми не можемо спостерігати потрібний рівень.

Моделі часткового коригування (анг. partial-adjustment model) допомагають нам змоделювати цю ситуацію.

Модель часткового коригування

Приклад

Ми хочемо знати, як змінюється бажана кількість сигарет, \(\color{#e64173}{\widetilde{\text{Cig}}_t}\), залежно від податку на сигарети поточного періоду, наприклад,

\[ \begin{align} \color{#e64173}{\widetilde{\text{Cig}}_t} = \beta_0 + \beta_1 \text{Tax}_t + u_t \tag{A} \end{align} \]

Уявіть, що фактичне вживання сигарет, \(\color{#6A5ACD}{\text{Cig}_t}\), не змінюється відразу (наприклад, стійкість звички). Натомість вживання залежить від поточного рівня бажань і попереднього рівня споживання

\[ \begin{align} \color{#6A5ACD}{\text{Cig}_t} = \lambda \color{#e64173}{\widetilde{\text{Cig}}_t} + \left( 1-\lambda \right) \color{#FFA500}{\text{Cig}_{t-1}} \tag{B} \end{align} \]

Модель часткового коригування

Продовження…

\[ \begin{align} \color{#e64173}{\widetilde{\text{Cig}}_t} &= \beta_0 + \beta_1 \text{Tax}_t + u_t \tag{A} \\[0.3em] \color{#6A5ACD}{\text{Cig}_t} &= \lambda \color{#e64173}{\widetilde{\text{Cig}}_t} + \left( 1-\lambda \right) \color{#FFA500}{\text{Cig}_{t-1}} \tag{B} \end{align} \]

Заміна \(\color{#e64173}{\widetilde{\text{Cig}}_t}\) з \((\text{A})\) на \((\text{B})\) дає

\[ \begin{align} \color{#6A5ACD}{\text{Cig}_t} &= \lambda \left( \beta_0 + \beta_1 \text{Tax}_t + u_t \right) + \left( 1-\lambda \right) \color{#FFA500}{\text{Cig}_{t-1}} \\[0.3em] &= \lambda\beta_0 + \lambda\beta_1 \text{Tax}_t + \left( 1-\lambda \right) \color{#FFA500}{\text{Cig}_{t-1}} + \lambda u_t \tag{C} \end{align} \]

Рівняння в \((\text{C})\) це ADL(1, 0).

Ми також можемо оцінити/відновити коефіцієнт швидкості налаштування \(\lambda\).

МНК у часових рядах

Незміщені коефіцієнти

Як і раніше, незміщеність МНК залежатиме від нашого припущення екзогенності, тобто, \(\mathop{\boldsymbol{E}}\left[ u_t \middle| X \right] = 0\).

Ми можемо розділити це припущення на дві частини.

  1. Збурення \(u_t\) не залежить від пояснювальних змінних у той самий період (тобто, \(X_t\)).
  1. Збурення \(u_t\) не залежить від пояснювальних змінних в інших періодах (тобто, \(X_s\) для \(s\neq t\)).

Нам потрібно, щоб обидві ці частини відповідали дійсності, щоб МНК був незміщеним.

Незміщені коефіцієнти

Нам потрібні обидві частини нашого припущення про екзогенність, щоб МНК був незміщеним:

\[ \begin{align} \mathop{\boldsymbol{E}}\left[ \hat{\beta}_1 \middle| X \right] &= \beta_1 + \mathop{\boldsymbol{E}}\left[ \dfrac{\sum_t \left( x_t - \overline{x} \right) u_t}{\sum_t \left( x_t - \overline{x} \right)^2} \middle| X \right] \end{align} \]

Тобто, щоб гарантувати, що чисельник дорівнює нулю, нам потрібно \(\mathop{\boldsymbol{E}}\left[ u_t | X \right] = 0\)—для обох \(\mathop{\boldsymbol{E}}\left[ u_t | X_t \right] = 0\) і \(\mathop{\boldsymbol{E}}\left[ u_t | X_{s} \right] = 0\) \((s\neq t)\).

Друга частина нашого припущення екзогенності—вимагає, щоб \(u_t\) був незалежним від усіх регресорів в інші періоди—не працює з динамічними моделями з змінними результату з відставанням.

Таким чином, МНК є упередженим для динамічних моделей із змінними результатів із запізненням.

Незміщені коефіцієнти

Щоб зрозуміти, чому динамічні моделі з лаговими змінними порушують наше припущення про екзогенність, розглянемо два періоди нашої простої моделі ADL(1, 0).

\[ \begin{align} \color{#e64173}{\text{Births}_t} &= \beta_0 + \beta_1 \text{Income}_t + \beta_2 \text{Births}_{t-1} + \color{#e64173}{u_t} \tag{1}\\[0.3em] \text{Births}_{t+1} &= \beta_0 + \beta_1 \text{Income}_{t+1} + \beta_2 \color{#e64173}{\text{Births}_t} + u_{t+1} \tag{2} \end{align} \]

У \((1)\) \(\color{#e64173}{u_t}\) чітко корелює з \(\color{#e64173}{\text{Births}_t}\).

Проте \(\color{#e64173}{\text{Births}_t}\) є регресором у \((2)\) (залежна лагова змінна).

∴ Збурення в \(t\) \(\left(\color{#e64173}{u_t}\right)\) корелює з регресором в \(t+1\) \(\left(\color{#e64173}{\text{Births} _t}\right)\).

Ця кореляція порушує другу частину нашої вимоги екзогенності.

Послідовні коефіцієнти

Ще не все втрачено.

Щоб МНК був послідовним, нам потрібена лише одночасна екзогенність.

Одночасна екзогенність: кожне порушення не корельовано з пояснювальними змінними за той самий період, тобто,

\[ \begin{align} \mathop{\boldsymbol{E}}\left[ u_t \middle| X_t \right] = 0 \end{align} \]

З одночасною екзогенністю оцінки МНК для коефіцієнтів у моделі часових рядів узгоджуються.

Послідовні коефіцієнти

Щоб зрозуміти, чому МНК узгоджується з одночасною екзогенністю, розгляньте оцінку МНК для \(\beta_1\) у

\[ \begin{align} \text{Births}_t &= \beta_0 + \beta_1 \text{Births}_{t-1} + u_t \end{align} \]

який ми показали, можна записати

\[ \begin{align} \hat{\beta}_1 &= \beta_1 + \dfrac{\sum_t \left( \text{Births}_{t-1} - \overline{\text{Births} } \right)u_t}{\sum_t \left( \text{Births}_{t-1} - \overline{\text{Births} } \right)^2} \end{align} \]

Послідовні коефіцієнти

\[ \begin{align} \mathop{\text{plim}} \hat{\beta}_1 &= \mathop{\text{plim}} \left( \beta_1 + \dfrac{\sum_t \left( \text{Births}_{t-1} - \overline{\text{Births} } \right)u_t}{\sum_t \left( \text{Births}_{t-1} - \overline{\text{Births} } \right)^2} \right) \\[0.3em] &= \beta_1 + \dfrac{\mathop{\text{plim}} \left[\sum_t \left( \text{Births}_{t-1} - \overline{\text{Births} } \right)u_t/T\right]}{\mathop{\text{plim}} \left[\sum_t \left( \text{Births}_{t-1} - \overline{\text{Births} } \right)^2/T\right]} \\[0.3em] &= \beta_1 + \dfrac{\color{#e64173}{\mathop{\text{Cov}} \left( \text{Births}_{t-1},\, u_t \right)}}{\mathop{\text{Var}} \left( \text{Births}_{t} \right)} \end{align} \]

\(\hspace{8em}=\beta_1\hspace{1em}\) if \(\color{#e64173}{\mathop{\text{Cov}} \left( \text{Births}_{t-1},\, u_t \right)=0}\)

.hi[Одночасна екзогенність] дає нам \(\color{#e64173}{\mathop{\text{Cov}} \left( \text{Births}_{t-1},\, u_t \right)=0}\) .

Послідовні коефіцієнти

Таким чином, якщо ми припустимо сучасну екзогенність, OLS є узгодженим для коефіцієнтів, навіть для моделей із лаговими залежними змінними.
